Explore Endpoints

What Is an API?

An API (Application Programming Interface) is a set of rules that enables two software applications to communicate via HTTP. A RESTful API supports Create, Read, Update, and Delete operations on server data from anywhere on the web.

Our Items API exposes endpoints to manage a collection of “items.” Here’s what you can do:

  • GET a list of all items or fetch a single item by ID.
  • POST a new item by sending a JSON payload.
  • PUT/PATCH an existing item using JSON.
  • DELETE an item you no longer need.

Endpoints at a Glance

📦 List All Items

GET /api/items/
Returns an array of all items in JSON format.

Try it now →
🔍 Retrieve Single Item

GET /api/items/{id}/
Replace {id} with the item’s unique ID to fetch it.

Try with ID=1 →
➕ Create New Item

POST /api/items/
Submit a JSON body (e.g. {"name":"New Item","description":"..."}) to add a new item.

Use curl or Postman →
✏️ Update Item

PUT /api/items/{id}/ or PATCH /api/items/{id}/
Replace {id} with the item’s ID and send JSON to modify.

Try with ID=1 →
🗑️ Delete Item

DELETE /api/items/{id}/
Permanently remove an item by its ID.

Try with ID=1 →

Quick Start (JavaScript Fetch)

Enter an Item ID below (or leave blank to fetch all). Click “Run” to see live results. 


// Example JS snippet:
fetch('https://items-api-6voy.onrender.com/api/items/1/')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
